# SenSante

Assistant de pre-diagnostic medical pour le Senegal.

## Description

SenSante utilise le Machine Learning pour aider au pre-diagnostic
des maladies courantes a partir des symptomes du patient.

## Structure du projet

- `data/` : donnees patients (CSV)
- `models/` : modele ML serialize
- `api/` : API FastAPI
- `frontend/` : interface web
- `notebooks/` : scripts d'exploration
